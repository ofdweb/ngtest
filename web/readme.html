Инструкция по развертыванию

  1. Поместить код приложения в той директории, где оно будет развернуто. Либо скопировать вручную, либо клонировать с GIT.
  2. Выполнить команду в корневой директории проекта 
    composer update
    Если менеджер пакетов не установлен, то скачать и установить его.
  3. Выполнить команду в корневой директории проекта 
    ./init
  4. В файле config/db.php задать параметры доступа к базе данных
  5. Выполнить команду для создания базы данных в корневой директории проекта 
    ./yii migrate/up
  6. В директории web создать директорию temp
  7. Перейти по пути web/atd3 относительно корневой директории проекта
  8. Выполнить команду 
    npm install
  9. Выполнить команду 
    gulp build --module=media
  10. Для виртуального хоста необходимо указать следующий DocumentRoot: путь_к_проекту/web
